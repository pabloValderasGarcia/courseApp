<h1>Basic Course App</h1>

<b>Frontend:</b>

- react-router
- react-redux
- redux-toolkit

<b>Backend:</b>

- mongoose (discriminators, populate)
- express (middleware)
---
<h1>Deployment</h1>

To deploy server

```bash
  cd server
  npm i
  npx ts-node app.ts
```

To deploy client

```bash
  cd client
  npm i
  npm run dev
```
